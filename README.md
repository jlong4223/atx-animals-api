# ATX Animals API

This is the backend API for ATX-Animals Client.

### Technologies Used:

- Node.js
- Express.js
- MongoDB
- Mongoose
- Mocha.js
- Chai.js
- Cors
- Morgan
- Postman
- jsonwebtoken
- Token based auth
- bcrypt
- Nodemailer
- OAuth2
- Node-Cron

### Future enchancments

- multer and/or cloudinary for img upload
