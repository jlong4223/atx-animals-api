# ATX Animals API

This is the backend API for ATX-Animals Client.

### Technologies Used:

- Node.js
- Express.js
- MongoDB
- Mongoose
- Mocha.js
- Chai.js
- Husky
- Cors
- Morgan
- Postman
- jsonwebtoken
- Token based auth
- bcrypt
- Nodemailer
- OAuth2
- Node-Cron
- Heroku

### Deployed API:

[Click](https://atx-animals-api.herokuapp.com/) to view the deployed API. Some routes are not accessible to the public.
